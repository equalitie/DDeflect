## Installation
1. [Install PhantomJS](http://phantomjs.org/download.html) on your system.  
2. Run ```npm install```  
3. Run ```node bundler```  
4. Try going to [```http://localhost:3000/?url=https://crypto.cat```](http://localhost:3000/?url=https://crypto.cat) as a test.
